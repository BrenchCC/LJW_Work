## 1. [cover] SubspaceAD：从正常子空间到特征投影
副标题：服务于工业零样本缺陷检测中的特征子空间投影任务
场景：mentor 论文汇报 / 2026-04-27
一句话：这篇论文用 DINOv2 patch 特征 + PCA 正常子空间，把异常检测简化为“投影后看残差”。
视觉建议：干净白底，左侧标题，右侧是电池缺陷图像被编码成点云并投影到低维平面的抽象图。

## 2. [content] 为什么这篇论文值得看
任务背景：新产品 / 第六款电池上线时，缺陷样本无标签，直接复用旧型号缺陷标签会遇到 domain shift。
核心矛盾：物理缺陷本质相似，但材质、光照、背景、纹理会改变成像域。
SubspaceAD 的价值：它证明“强视觉特征 + 轻量统计子空间”可以在少样本甚至极少标注条件下产生稳定异常分数。
对本任务的启发：先把缺陷判别从原始图像空间转到特征空间，再在子空间里做投影、距离和残差判断。

## 3. [content] 背景任务：零样本缺陷检测的子空间假设
工业目标：对无标签新电池图像，直接判断划痕 / 正常，减少标注周期。
可用资源：旧型号电池已有缺陷样本和标签，新型号只有产线采集图像。
关键假设：划痕的底层物理特征可迁移，产品外观域变化需要被剥离或弱化。
子空间方法：用预训练 ResNet / ViT 抽高维特征，再学习共享缺陷子空间，让同类缺陷聚拢、异类样本分离。
判别方式：新样本投影到子空间后，用距离度量或轻量分类器输出缺陷概率。

## 4. [content] SubspaceAD 方法：正常子空间 + 残差评分
输入：k = 1, 2, 4 张正常训练图像，以及一张测试图像。
特征：冻结 DINOv2-G，提取 patch token，并聚合第 22-28 层中间特征。
建模：对正常 patch 特征拟合 PCA，保存均值向量 mu 和主子空间基 C。
投影：x_proj = mu + C C^T (x - mu)。
评分：S(x) = ||x - x_proj||_2^2，残差越大越异常。
视觉建议：左侧是正常 patch 点云形成低维平面，右侧是异常点投影后留下明显垂直残差。

## 5. [data] 实验结论：简单投影能打到强基线
MVTec-AD 1-shot：图像级 AUROC 97.1%，像素级 AUROC 97.5%。
VisA 1-shot：图像级 AUROC 93.4%，像素级 AUROC 98.2%。
相较 AnomalyDINO：VisA 1-shot 图像级 AUROC 提升约 6.0 个百分点，PRO 提升约 1.0 个百分点。
定性表现：异常热图更贴近 GT 区域，背景误激活更少。
谨慎结论：SubspaceAD 大多数指标领先，但并非每个单项都绝对第一。

## 6. [content] 关键消融：子空间不是越大越好
PCA 解释方差阈值 tau = 0.99 是核心设置。
tau 在 0.95 到 0.99 时性能稳定；tau = 1.00 时性能明显下降。
解释：如果保留完整特征空间，异常 patch 也会被重构掉，残差信号被削弱。
分辨率：448 px 以上较稳，672 px 是跨数据集折中。
backbone：DINOv2-G 显著优于小模型，说明性能上限主要来自基础特征质量。

## 7. [content] 和我们的特征子空间投影任务如何对齐
一致点 1：都把工业缺陷判断从图像空间迁移到特征空间。
一致点 2：都依赖预训练视觉 backbone 提供可迁移的局部表征。
一致点 3：都希望通过投影后的距离 / 残差，降低对人工标签和复杂训练的依赖。
差异点 1：SubspaceAD 建模的是“正常子空间”，我们的背景任务更偏“共享缺陷子空间”。
差异点 2：SubspaceAD 需要少量正常样本；背景任务希望新型号零标注即可投影判断。
差异点 3：SubspaceAD 用 PCA 闭式解；背景任务可能需要 UDA 的聚拢 / 排斥目标来对齐跨型号域。

## 8. [content] 下一步：把论文改造成可验证路线
Step 1：用 DINOv2 / ViT 提取旧型号与新型号 patch 或 region 特征。
Step 2：分别构建两个 baseline：正常 PCA 残差；缺陷原型子空间投影。
Step 3：评估距离策略：欧氏距离、余弦相似度、Mahalanobis 距离和轻量 SVM / MLP。
Step 4：加入域对齐目标：同类缺陷聚拢、背景域排斥、正常区域残差校准。
Step 5：用可视化检查子空间：投影前后点云、异常热图、阈值敏感性和跨型号泛化。
